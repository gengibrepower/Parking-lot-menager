AMPS

automated management parking system

O que somos?

O AMPS é um modelo para gerenciar qualquer estacionamento de forma otimizada. Utilizando de um modelo matemático ele direciona os carros que entrarem no estacionamento para a vaga ideal evitando superlotação em apenas uma área e exagerado de procura de uma vaga.
